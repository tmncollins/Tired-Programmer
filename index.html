NEW DOCUMENTATION : FENWICK TREE

How to Implement Fenwick Tree Data Structures, April 27 2022

Ever heard of a Fenwick Tree? How about a Binary Indexed Tree? Sounds complicated, right? Well, maybe not!

My second short documentation is out, describing how to use and implement a very useful data structure: a Fenwick Tree.

Fenwick Trees are efficient structures to quickly update and query ranges.

You can find the documentation in the Problem Library under Segment Trees.

I am hoping to release more documentation on other data structures and algorithms in the future.

Don't forget to say if you find any errors!
